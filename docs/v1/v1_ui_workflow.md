## 2. 7阶段操作流程

### 2.1 启动

| 步骤 | 操作 | 系统响应 |
|------|------|---------|
| 1 | 执行 `python run_app.py` | 加载默认配置 + user_tuning覆盖 |
| 2 | — | 构建设备卡片、选择器、预览面板、日志面板 |
| 3 | — | 界面就绪，状态"空闲" |

### 2.2 配置

| 步骤 | 操作 | 系统响应 |
|------|------|---------|
| 1 | 编辑设备串口参数 | 更新设备配置 |
| 2 | 选择路由模式 | 更新运行范围 |
| 3 | 选择温度/CO2/压力范围 | 更新点位预览 |
| 4 | 点击"加载配置" | 重新加载配置文件 |

### 2.3 自检

| 步骤 | 操作 | 系统响应 |
|------|------|---------|
| 1 | 点击"自检"按钮 | 启动独立线程执行连接性检查 |
| 2 | — | 构建临时设备实例，尝试连接 |
| 3 | — | 显示设备状态报告（成功/失败/超时） |
| 4 | — | 关闭临时设备，释放串口 |

### 2.4 运行

| 步骤 | 操作 | 系统响应 |
|------|------|---------|
| 1 | 点击"启动"按钮 | 检查启动就绪条件 |
| 2 | — | 构建运行时配置 + 设备实例 |
| 3 | — | 启动后台worker线程执行 CalibrationRunner.run() |
| 4 | — | 锁定控件（禁止配置修改/自检/阀门控制） |

### 2.5 监控

| 步骤 | 操作 | 系统响应 |
|------|------|---------|
| 1 | — | 轮询IO日志文件解析最新设备值 |
| 2 | — | 推断路由状态，更新设备卡片 |
| 3 | — | 更新进度状态和日志显示 |

### 2.6 停机

| 步骤 | 操作 | 系统响应 |
|------|------|---------|
| 1 | 点击"停止"或流程自然结束 | 请求runner停止 |
| 2 | — | 等待worker线程结束 |
| 3 | — | 执行安全停机（恢复基线） |
| 4 | — | 解锁控件，导出结果 |

### 2.7 后处理

| 步骤 | 操作 | 系统响应 |
|------|------|---------|
| 1 | 执行 `python run_v1_postprocess.py` | 启动离线后处理GUI |
| 2 | 执行 `python run_v1_merged_sidecar.py --run-dir <dir>` | 启动合并侧车 |

## 3. 界面状态机

### 3.1 状态定义

| 状态 | 说明 | 控件状态 |
|------|------|---------|
| 空闲 | 初始状态，等待用户操作 | 全部可操作 |
| 自检中 | 自检线程运行中 | 启动/自检禁用 |
| 运行中 | 校准worker线程运行中 | 大部分控件锁定 |
| 暂停 | 流程暂停（当前V1不支持暂停） | 部分控件锁定 |
| 停机中 | 安全停机执行中 | 全部禁用 |
| 错误 | 运行异常中止 | 部分解锁，可重试 |

### 3.2 合法转换

```
空闲 ──启动自检──▶ 自检中 ──自检完成──▶ 空闲
空闲 ──启动运行──▶ 运行中 ──正常结束──▶ 停机中 ──停机完成──▶ 空闲
运行中 ──异常中止──▶ 错误 ──确认──▶ 空闲
运行中 ──用户停止──▶ 停机中 ──停机完成──▶ 空闲
空闲 ──恢复基线──▶ 停机中 ──停机完成──▶ 空闲
```

### 3.3 锁定判断

`_controls_locked()`（app.py:9210-9215）：

```python
locked = safe_stop_in_progress
      or startup_thread.is_alive()
      or worker.is_alive()
```

## 4. 安全约束

### 4.1 运行中禁止操作

| 操作 | 禁止原因 | 日志消息 |
|------|---------|---------|
| 自检 | 与运行线程冲突 | "流程运行中，禁止执行自检" |
| 阀门控制 | 与运行线程冲突 | "流程运行中，无法打开阀门控制页面" |
| 湿度发生器页面 | 与运行线程冲突 | "流程运行中，无法打开湿度发生器页面" |
| 露点仪页面 | 与运行线程冲突 | "流程运行中，无法打开露点仪页面" |
| 测温仪页面 | 与运行线程冲突 | "流程运行中，无法打开测温仪页面" |
| 配置修改 | 避免运行时配置变更 | 控件禁用 |

### 4.2 关闭窗口安全停机

用户关闭窗口时：

1. 检查 `worker.is_alive()`
2. 若运行中：弹出确认对话框
3. 确认后：请求停止 → 等待线程结束 → 安全停机 → 退出
4. 超时：日志提示，不强制退出

### 4.3 真机执行双门禁

真机在线工程验证必须同时满足：

1. CLI 传入 `--real-device`
2. 环境变量 `ALLOW_REAL_DEVICE_WRITE=1`
